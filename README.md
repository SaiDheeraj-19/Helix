## 🚀 About Helix

Helix is a next-generation open-source software development project management tool. It provides a simple, dynamic, and beautiful interface to help teams plan, track, and manage their work effortlessly. 

With a focus on a highly polished UI, premium aesthetics, and responsive performance, Helix is designed to elevate your project management experience.

## ✨ Features

- **Premium Interface**: A meticulously designed aesthetic with dynamic interactions and sleek workflows.
- **Modern Tracking**: Easy-to-use boards, lists, and timelines for agile project management.
- **Streamlined Onboarding**: A seamless setup process to get you and your team started instantly.
- **Highly Customizable**: Adapt the tool to fit your team's unique workflows and preferences.
- **Open Source**: Built on transparent and extensible foundations.

## 🛠️ Tech Stack

- **Frontend**: Next.js, React, Tailwind CSS
- **Backend**: Django, PostgreSQL, Redis
- **State Management**: MobX
- **Deployment**: Docker

## 🚀 Getting Started

### Prerequisites

- Docker & Docker Compose
- Node.js (v18+)
- pnpm

### Local Development

To run Helix locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SaiDheeraj-19/Helix.git
   cd Helix
   ```

2. **Run the setup script:**
   ```bash
   ./setup.sh
   ```

3. **Start the development servers:**
   ```bash
   pnpm dev
   ```
   The web application will be available at `http://localhost:3000`.
